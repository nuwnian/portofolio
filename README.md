## 🚀 Quick Start

### Local Development

1. **Clone the repository**
   ```bash
   git clone [repository-url]
   cd portfolio
   ```

2. **Open in VS Code**
   ```bash
   code .
   ```

3. **Launch with Live Server**
   - Install VS Code Live Server extension
   - Right-click `index.html` → "Open with Live Server"
   - Portfolio opens at `http://localhost:5500`

### GitHub Pages Deployment

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Initial portfolio setup"
   git push origin main
   ```

2. **Enable GitHub Pages**
   - Go to repository Settings
   - Navigate to Pages section
   - Select "Deploy from branch" → `main`
   - Site available at `https://username.github.io/repository-name`

## 🎯 Customization Guide

### Personal Information
Update the following in `index.html`:
- Hero section name and description
- About section content
- Contact information and social links
- Project details and links

### Styling
Modify `style.css` custom properties:
```css
:root {
    --primary-color: #6c5ce7;
    --secondary-color: #fdcb6e;
    --accent-color: #00b894;
    /* ... other variables */
}
```

### Projects
Add new projects in the projects section:
- Update project cards with real screenshots
- Add live demo and GitHub links
- Include technology tags and descriptions

### Contact Form
Integrate with a backend service:
- Replace form handling in `script.js`
- Add email service integration (EmailJS, Formspree, etc.)
- Update success/error messages
